
# 冷丶布丁的个人空间 - 本地开发与服务器部署指南

本指南将协助您将 **冷丶布丁的个人空间** 项目从零开始部署到您的本地环境及远程服务器。

## 1. 准备工作

在开始之前，请确保您的服务器或本地环境已安装以下软件：
*   **Node.js** (建议 v18.0.0 或更高版本)
*   **Nginx** 或 **Apache** (用于服务器端托管静态文件)
*   **Git** (用于版本管理)

---

## 2. 关键文件修改 (部署前必看)

由于本项目集成了 Google Gemini AI 以及预留了后端接口，在部署前，您**必须**检查并修改以下文件：

### A. Gemini API 配置 (`pages/ToolDetail.tsx`)
项目中 AI 智能引擎使用了 `process.env.API_KEY`。
*   **修改点**：在本地测试时，您可以直接将 API Key 字符串暂时替换到代码中（不推荐发布），或者通过构建工具注入。
*   **获取 Key**：请访问 [Google AI Studio](https://aistudio.google.com/) 获取。

---

## 3. 本地部署步骤

1.  **下载代码**：
    将项目文件保存到本地目录。

2.  **安装依赖**：
    在项目根目录下运行：
    ```bash
    npm install
    ```

3.  **启动开发服务器**：
    ```bash
    npm run dev
    ```
